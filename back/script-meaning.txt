--respawn: Этот флаг указывает ts-node-dev перезапускать весь процесс при изменениях в файлах, а не только перекомпилировать и перезагрузить только измененные файлы. Это полезно, если изменения в файлах требуют полного перезапуска приложения для корректной работы.

--transpile-only: Этот флаг отключает типизацию TypeScript при трансляции, что ускоряет процесс компиляции. Ошибки типизации будут игнорироваться, и код будет транспилироваться без проверки типов. Обычно используется на этапе разработки для повышения скорости.

--exit-child: Этот флаг завершает дочерние процессы при перезапуске. Это помогает предотвратить утечки памяти и проблемы с неочищенными ресурсами, которые могут возникнуть при частых перезапусках приложения.
